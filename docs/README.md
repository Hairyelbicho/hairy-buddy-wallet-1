# ?? Hairy Wallet - Web Oficial

Bienvenido al repositorio de la **web oficial de Hairy Wallet**, el ecosistema descentralizado y seguro para proteger tus criptomonedas, apoyar a los animales y gestionar tu wallet con IA integrada.

## ?? Contenido

Este repositorio contiene los archivos generados en la carpeta `dist` por Vite, listos para ser desplegados en GitHub Pages o cualquier servidor estático:

- `index.html`
- Carpeta `assets/`
- `favicon.ico`
- `placeholder.svg`
- `robots.txt`

## ?? Despliegue

Puedes desplegar esta web directamente en:
- GitHub Pages
- Vercel
- Netlify
- Servidor propio (como `hairyelbicho.com`)

### ? Requisitos para despliegue

1. Asegúrate de subir el contenido de la carpeta `dist/`.
2. Si usas GitHub Pages, ve a la configuración del repositorio y selecciona como fuente `/dist`.

## ?? Política de Privacidad

La política está disponible en la ruta `/privacidad` de la web. También puedes incluirla manualmente en tu servidor si es necesario.

## ?? Proyecto Hairy

Esta web forma parte del ecosistema **Hairy**, cuyo objetivo es combinar tecnología blockchain con impacto social para proteger animales, educar en finanzas y facilitar herramientas digitales seguras.
